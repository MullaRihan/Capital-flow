# Capital Flow 

A personal expense tracker built with HTML, CSS and JavaScript.

## Features
- Track income and expenses
- Visual charts with Chart.js
- Data persists with localStorage
- Category breakdown
- Recent transactions table

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js
